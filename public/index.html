Word Scramble

This is a simple JSON api to descramble strings into English words. It is written in Grape, and you can see the source here.

Give it a try!

Try realapin, liopts, or otauminn

JSON Api

You can access the api at /descramble/:scrambled_string(.json) You'll get back an array of matching words.

Example

curl http://smooth-stream-5874.herokuapp.com/descramble/liopts.json

# => ["pilots", "spoilt", "pistol"]
